== Description ==

A plugin that automatic download and resize your images on Post when it larger than your width and height set on Option page.

This plugin is base on ImageScaler by PaulButler, I only change some code for the better work.

== Installation ==

1. Upload `imagescaler.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. It should work when you save your post.

== Frequently Asked Questions ==

= My images on old post doesn't resize =

Edit your post and save, that plugin only work when you save your post.

= I can't save my post. It's take forever to loading =

Report on my plugin's page.
